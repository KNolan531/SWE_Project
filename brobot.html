                 Baseball Robot - Mobile App - Prototype


This is a project I am currently working on for course SWE 4324 - User Centered Design at Kennesaw State University.

This project involved teaming up with another student (the user) and using an iterative process to gather requirements and functionality for a mobile app for a robot.

The robot is a baseball themed droid that performs various baseball training operations like pitching and hitting.

For this project we are using Marvel, a prototyping development system to create an interactive app.
